ตัวอย่างเมนูด้านล่างแบบ Sticky

เลื่อนหน้าเว็บเพื่อดูว่าเมนูด้านล่างยังคงลอยอยู่ในตำแหน่งเดิม

หลักการทำงาน

position: fixed - คุณสมบัติหลักที่ทำให้เมนูลอยอยู่ในตำแหน่งเดิม

bottom: 0 - กำหนดให้ติดด้านล่างของหน้าจอ

z-index: 1000 - ให้เมนูอยู่ด้านบนสุดเสมอ

ฟีเจอร์เพิ่มเติม

  • รองรับ Responsive Design
  • เอฟเฟกต์ Ripple เมื่อคลิก
  • Badge สำหรับจำนวนสินค้า
  • Transition แบบ Smooth
  • รองรับ Safe Area สำหรับมือถือ

CSS ที่สำคัญ

.bottom-navigation {
    position: fixed;    /* คีย์หลัก */
    bottom: 0;          /* ติดด้านล่าง */
    left: 0;
    right: 0;
    width: 100%;
    z-index: 1000;      /* อยู่ด้านบนสุด */
}

JavaScript Functionality

JavaScript จัดการ:

  • การเปลี่ยน Active State
  • การนำทางระหว่างหน้า
  • การอัพเดท Badge
  • เอฟเฟกต์ต่าง ๆ
